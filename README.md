# Wishful App 

A mobile and web application that allows users to share their wishes and dreams, connect with others, and help make them come true. The app inspires generosity and encourages people to fulfill small, meaningful wishes.

## Demo
You can try out the app live here:  
[Wishful App Live Demo](https://farhad5351.github.io/wishful-app/)

## Features
- **Post Wishes:** Users can share their personal wishes or dreams with the community.
- **Social Interaction:** Users can interact with others' wishes, provide support, or even help make them come true.
- **Mobile and Web Support:** The app is designed as a Progressive Web App (PWA), meaning it can be installed and used on both mobile and desktop.
- **Google Authentication:** Secure sign-in with Google accounts to easily join and interact with the community.
